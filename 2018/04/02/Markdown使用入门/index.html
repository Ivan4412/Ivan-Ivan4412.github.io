Markdown使用入门

  2018-04-02    Markdown

Markdown使用入门：

1、特殊字符：

对于部分Markdown语法中的特殊字符，需输出原字符的话，使用反斜线：即可。 如： ©：\©

2、换行

上一行行尾至少有两个及以上的空格

3、标题：

  1. 类 Setext 形式：

    代码: 

    This is an H1  
=============  
This is an H2  
-------------

    输出：

    This is an H1

    This is an H2

  1. 类 Atx 形式:

    代码: 

    # 这是 H1  
## 这是 H2

    输出：

    这是 H1

    这是 H2

4、区块引用

每行的最前面加上 >。
代码： 

> This is a blockquote with two paragraphs.

输出：

This is a blockquote with two paragraphs.

多重嵌套：
代码： 

> > This is nested blockquote.
>
> Back to the first level.

输出：

This is nested blockquote.

Back to the first level.

嵌套其他 Markdown 语法：
代码： 

> ## 这是一个标题。
> 
> 1.   这是第一行列表项。
> 2.   这是第二行列表项。
> 
> 给出一些例子代码：
> 
>     return shell_exec("echo $input | $markdown_script");

输出：

这是一个标题。

  1. 这是第一行列表项。
  2. 这是第二行列表项。

给出一些例子代码：

return shell_exec("echo $input | $markdown_script");

5、列表

  1. 无序列表使用星号、加号或是减号作为列表标记：
    代码： 

    * Red
* yellow
+ Green
- Blu

    输出：

    • Red
    • yellow
    • Green
    • Blu

  2. 有序列表则使用数字接着一个英文句点：(表标记上使用的数字并不会影响输出结果)
    代码： 

    2.  Bird
1.  McHale
3.  Parish

    输出：

    1. Bird
    2. McHale
    3. Parish

  3. 列表中加入引用，需用> 缩进：
    代码： 

    +   A list item with a blockquote:  
> This is a blockquote
> inside a list item.

    输出：

    • A list item with a blockquote:

      This is a blockquote
      inside a list item.

6、代码

  1. 代码区块：
    缩进 4 个空格或是 1 个制表符。 

    这是一个代码区块。

  2. 标记小段行内代码：
    使用反引号把它包起来（`）,对于代码中的`，使用两个反引号括住（`` ) 。
    代码：

    Use the `printf()` function.
``There is a literal backtick (`) here.``

    输出：
    Use the printf() function.
    There is a literal backtick (`) here.

7、分割线

一行中用三个以上的星号、减号、底线来建立一个分隔线。
代码： 

* * *
---

输出：

8、链接

  1. 行内式链接：
    方块括号后面紧接着圆括号并插入网址链接，后面紧接双引号包起的文字可以显示为链接提示文字。
    代码： 

    [This link](https://ivan4412.github.io/ "Ivan的博客") has no title attribute.

    输出:
    This link has no title attribute.

  2. 参考式链接：
    在文中任意地方使用
    [id]: + 链接地址
    格式记录链接地址。
    对于需链接文字，在链接文字括号后再接一个方括号，括号中内容为链接的标签id
    代码：

    This is [an example][ID1] reference-style link.  

[ID1]: https://ivan4412.github.io/  "Ivan的博客"

    输出：
    This is an example reference-style link.

9、强调

使用**或__包住需强调的字词。
代码： 

*single asterisks*  
_single underscores_

输出：
single asterisks
single underscores

10、图片

图片链接方式和链接类似，亦分为行内式和参考式。

  1. 行内式：
    使用!开头，紧接方括号括住的图片替代文字，紧接普通括号括住网址
    代码：

    ![Ivanlogo](https://ivan4412.github.io/img/logo.jpg "Ivan博客Logo")

    输出：

  2. 参考式：
    使用!开头，紧接方括号括住的图片替代文字，紧接方括号括住的地址标签id。
    代码：

    ![Ivanlogo][img1]

    输出：
